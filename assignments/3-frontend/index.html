Di dunia komputer, “Backend” mengacu pada bagian mana pun dari situs web atau program perangkat lunak yang tidak dilihat pengguna. Ini kontras dengan frontend, yang mengacu pada antarmuka pengguna program atau situs web. Dalam terminologi pemrograman, backend adalah “lapisan akses data,” sedangkan frontend adalah “lapisan presentasi.” Sebagian besar situs web modern adalah dinamis, artinya konten halaman web dihasilkan secara langsung. Halaman dinamis berisi satu atau lebih skrip yang berjalan di server web setiap kali halaman diakses. Script ini menghasilkan konten halaman, yang dikirim ke browser web pengguna. Segala sesuatu yang terjadi sebelum halaman ditampilkan di browser web adalah bagian dari backend. Contoh proses backend meliputi: Memproses permintaan halaman web yang masuk menjalankan skrip (PHP, ASP, JSP, dll.) Database adalah sekumpulan data yang dikelola berdasarkan ketentuan tertentu yang saling berkaitan sehingga memudahkan dalam pengelolaannya. Containers atau adalah sebuah media penyimpanan yang digunakan dalam proses pemindahan barang.
